💡 知识点
There is / There are 的用法
📌 什么时候用？
表示某处有某物
There is + 单数名词
There are + 复数名词
🔊 例句
There is a refrigerator in the kitchen. 厨房里有一台冰箱。
There is a table in the room. 房间里有一张桌子。
⚠️ 注意
There is a refrigerator ✅
It is a refrigerator ❌（表示"有"用 There is）
💡 知识点
方位介词
📌 描述位置
on the right 在右边
on the left 在左边
in the middle of 在……中间
🔊 更多方位
on the table 在桌子上
in the kitchen 在厨房里
near the window 在窗户附近
🎯 描述房间步骤
1. 先说 There is/are + 物品
2. 再说 It is/They are + 位置
📝 核心句型
There is... / It is on the...
有…… / 它在……
📌 句型结构
There is a/an + 名词 + 位置.
It is on the + 方位.
🔊 例句
There is a refrigerator in the kitchen.
It is on the right.
There is a table in the middle of the room.
✅ 完整描述
There is a cup on the table.
The cup is clean.（描述特征）
