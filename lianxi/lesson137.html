💡 知识点
第一条件句（真实条件句）
📌 构成
If + 一般现在时, 主句 + will/can/may + 动词原形
表示可能发生的条件和结果
if 从句用一般现在时，主句用一般将来时
📌 课文例句
If I win a lot of money, I will buy you a mink coat.
如果我赢了许多钱，我就给你买件貂皮大衣。
If we win a lot of money, we will travel round the world.
如果我们赢了很多钱，我们就去周游世界。
📌 if从句的位置
If 从句在前面，用逗号隔开：If..., ...
If 从句在后面，不用逗号：... if ...
If it rains, I will stay at home.
I will stay at home if it rains.
💡 知识点
重点词汇与表达
📌 depend on
依靠；取决于
It depends on the weather. 取决于天气。
It depends on you. 取决于你。
📌 spend...on
在……上花费
We spent a lot of money on the house.
我们在房子上花了很多钱。
📌 live on
靠……生活/维持生活
We won't have any money to live on.
我们没有钱维持生活。
📌 win / earn
win 赢得（比赛、奖金等）
earn 挣得（通过劳动获得）
We'll have to earn it first! 我们必须先挣到它！
📝 核心句型
If... will...
如果……就……（第一条件句）
📌 句型结构
If + S + V(一般现在时), S + will + V(原形)
S + will + V(原形) + if + S + V(一般现在时)
🔊 例句
If I win a lot of money, I will buy you a mink coat.
If we spend all the money, we will be poor again.
